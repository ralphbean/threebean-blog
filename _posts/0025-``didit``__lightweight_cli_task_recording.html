I wrote and published didit this afternoon. I hope you find it useful for the same reasons I wrote it.

$$code(lang=text, style=monokai) % didit-remember -c work -m 'Wrote `diddit`. Thank god.' % didit-remember --message 'Helped L. User parallelize his ``Mathematica`` code.' % didit-remember -c personal # <-- This launches `vim` for me! % didit-report --categories=work,general,personal Category 'work, general, personal' over timespan 'week' ------------------------------------------------------- ---- 2011-05-05: - Wrote `diddit`. Thank god. - Helped L. User parallelize his ``Mathematica`` code. - Drank a beer. $$/code

One of the upshots of using .rst:

$$code(lang=text, style=monokai) % didit-report --category=work > thisweek.rst && rst2pdf thisweek.rst $$/code
