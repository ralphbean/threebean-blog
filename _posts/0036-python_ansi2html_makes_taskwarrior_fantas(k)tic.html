Over a year ago, I wrote a python module called ansi2html that takes colored ansi as input and produces colored, formatted html. With it you can do something like the following: [sourcecode language="text"] $ ls --color=always | ansi2html > directory.html $ sudo tail /var/log/messages | ccze -A | ansi2html > logs.html [/sourcecode] Much more recently I discovered taskwarrior, a command line todo list manager that rocks. It features the ability to produce burndown charts of your todos over time. I use the following in a cronjob to produce an online version of my burndown chart. [sourcecode language="text"] $ task burndown | ansi2html | ssh me@web "cat > ~/static/burndown.html" [/sourcecode] My regularly-updated chart is hosted on threebean.org. If you come up with any neat uses for ansi2html, please let me know. I'd love to hear about them.
